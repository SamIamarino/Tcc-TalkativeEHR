## ✅ Recommended Stable Setup for ESP32

| Setting            | Value                        |
|--------------------|-----------------------------|
| **Upload Speed**   | `115200`                    |
| **Flash Mode**     | `DIO`                       |
| **Flash Frequency**| `80 MHz`                    |
| **Partition Scheme** | Default 4MB with SPIFFS   |
| **PSRAM**          | Disabled <br>*(unless your board has it)* |
